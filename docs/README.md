Leo Nicatot是一位计算机科学的博士生，至于他将来是什么他也不知道，反正当下他是计算机科学的博士生。

Leo Nicatot是一位经历过痛苦的博士生[^1]，至于他将来还会不会有更加痛苦的经历他也不知道，反正当下他经历过痛苦（所以他现在这个主页也没有之前那么轻松戏谑了）。

Leo Nicatot是一位慢性子且纠结的博士生[^2]，至于他将来还会不会这么慢性子且纠结他也不知道[^3]。

Leo Nicatot是一个没有在世俗标准上做出什么的loser，至于他将来在世俗标准上能不能做出什么他也不知道，反正他当下是个（世俗意义上的）loser。

Leo Nicatot是一个喜欢分享想法但是没什么人听的人，至于他将来分享的东西有没有愿意听他也不知道，反正他当下分享的东西没什么人愿意听。

Leo Nicatot是一个总是在关注未来的人，至于他将来会不会还是关注未来他也不知道，反正他当下一直在关注未来，难以关注当下就是了[^4]。
