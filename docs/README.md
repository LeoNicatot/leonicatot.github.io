# 关于Leo Nicatot

Nicatot是一个计算机科学的博士生，至于他将来是什么（假如那时候还活着的话）他也不知道，反正当下他是计算机科学的博士生。

Nicatot是一个难以感受到快乐的人[^1]，至于他将来能不能感受到快乐他也不知道，反正当下他难以感受到快乐（所以他现在这个主页也没有之前那么轻松戏谑了）。

Nicatot是一个慢性子且纠结的人[^2]，至于他将来还会不会这么慢性子且纠结他也不知道[^3]，反正他当下是个慢性子且纠结的人。

Nicatot是一个没有在世俗标准上做出什么的loser，至于他将来在世俗标准上能不能做出什么他也不知道，反正他当下是个（世俗意义上的）loser。

Nicatot是一个喜欢分享想法但是没什么人听的人，至于他将来分享的东西有没有愿意听他也不知道，反正他当下分享的东西没什么人愿意听。

Nicatot是一个总是在关注未来的人，至于他将来会不会还是关注未来他也不知道，反正他当下一直在关注未来，难以关注当下就是了[^4]。

[^1]: 在Nicatot的紫微命盘里，命宫是空宫（即没有主星），命里会被对面的迁移宫影响，而迁移宫中的天同自带一个生年忌。天同是主管快乐的星，它化忌了还冲进命宫，这很生动形象地描述了Leo的精神状态。

[^2]: Nicatot的命宫有一颗辅星陀罗（注：陀罗象征原地打转），由于命宫里没有主星坐镇，这颗陀罗直接翻身当家做主了，把它那种让人左右为难、磨磨唧唧的特性发挥到了极致，这也完美地描述了Nicatot在现实中纠结与磨叽的一面。

[^3]: 如果按紫微斗数的理论，Nicatot的后半生要看身宫。Nicatot的身宫落在福德宫（代表精神世界），福德宫里有擎羊（注：擎羊象征一把钢刀，主急躁和冲劲）。更惨的是，他的身宫同样没有主星坐镇，所以这把“刀”的威力也被放大了。这预示着他以后可能会从一个“慢吞吞的纠结怪”，进化成一个内心焦躁不安、脾气变烂、但依然在原地打转的矛盾体。

[^4]: Nicatot会这么说话显然是受到Eckhart Tolle和Jon Kabat-Zinn等人的影响。
